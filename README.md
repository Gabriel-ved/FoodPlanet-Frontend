# Foodplanet - frontend

A [Foodplanet](https://foodplanet-frontend.herokuapp.com/) e uma marca ficticia que simula uma plataforma de venda e compra de produtos.   

## Ferramentas usadas

- Reactjs
- Axios
- React dropzone
- React router dom

**A plataforma consome a api [Foodplanet - backend](https://github.com/Gabriel-ved/FoodPlanet-Backend)**
